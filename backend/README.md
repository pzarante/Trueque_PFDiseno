## Project setup

```bash
$ npm install
$ npm install axios
$ npm install --save-dev @types/axios
$ npm install dotenv
$ npm install @nestjs/config
```
Para ejecutar:
cd backend
npm start

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
