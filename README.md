# Trueque_PFDiseño

## Instalación

### Frontend
```
cd frontend
npm install
```

### Backend
```
cd backend
npm install
```

### Modelo NLP
```
cd modelo_NLP
pip install -r requirements.txt
```

## Ejecución

### Frontend
```
cd frontend
npm run dev
```

### Backend
```
cd backend
npm run dev
```

### Modelo NLP
```
cd modelo_NLP
python app.py
```
